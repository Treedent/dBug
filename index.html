dBug is a simple php 7.1+ class to debug PHP variables such as :

  • String
  • Number
  • Boolean
  • Null
  • Array
  • Mysql Query Single Result
  • Mysql Query Multiple Results
  • Object
  • JSON
  • XML
  • PostegreSQL resource
  • GD Image resource
  • Exif Image information
